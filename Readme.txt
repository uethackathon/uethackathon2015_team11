HighSchool Support

1.Thư viện sử dụng

- jdbc mysql driver - kết nối mysql
- tiles - tạo template cho website
- orgjson - encode , decode json
- activation, mail - gửi email
- pnotify.js - tạo thông báo notifi giống facebook 

2. Framework
	
	- Bootstrap - giao diện cho website
	- AngularJs - code phía client
	- Spring MVC - code phía server
	
3. Hướng dẫn cài đặt

	- Cài đặt Ivy : eclipse -> hepl -> eclipse marketplace -> gõ vào thanh tìm kiếm "ivy" -> cài đặt kết quả đầu tiên
	- Sau khi cài thành công ivy, import project vào trong eclipse
	- Click chuột phải vào project chọn ivy -> reload settings
	- Đợi ivy load các thư viện về

4. Cấu hình hệ thống
	
	- từ thư mục gốc của project -> chọn main/resource -> application.properties -> cấu hình lại database phù hợp với thông số của bạn

5. Cài đặt Tomcat-8
	- tải tomcat-8 từ trên mạng về vào preferences -> server -> runtime enviroment -> add -> chọn thư mục tomcat-8
